git - любая гит команда начинается с этого слова 
add - добавляет данный файл(ы) к текущему комиту
commit - вносит выбраные и добавленые файлы в историю git'a, использовать с -m "сообщение для описанию коммита"
push - отправить текущее состояние репозиторию в удаленное хранилище
status - статус, выводит информацию о состоянии данного гит репозитория
init - создать гит репозиторию в данной директории
